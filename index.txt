LEER ANTES DE INICIAR EL USO DEL PROGRAMA.

Informacion general:
En esta primera entrega, encontraras tres carpetas con toda la informacion, codigo y utilidades necesarias para entender y ejecutar el programa. Estas carpetas son las siguientes:

- DOCS: contiene un fichero .pdf con toda la documentacion del proyecto:
	+ Casos de uso y su explicacion
	+ Diagrama de clases y la descripcion de cada una
	+ ED + algoritmos utilizados
	+ Relacion clases/miembro

	- El manual de usuario
	- Los juegos de prueba
	- El diagrama de clases UML
	- entre otros mas

- FONTS: contiene todos los archivos .java del proyecto. Estos se separan entre "main". Siendo main la carpeta con el codigo que se ejecutara en el driver.

- EXE: contiene el esqueleto Gradle utilizado durante el proyecto. Es en este directorio donde podras ejecutar las diferentes funcionalidades del programa:
	+ Con el comando './gradlew run --console=plain' podras ejecutar la aplicacion (NOTA: si esta opcion falla, ejecuta primero './gradlew clean')
